Introduction#

The EUV Snapshot Imaging Spectrograph (ESIS) is a sounding rocket-based tomographic imaging spectrograph that was launched from White Sands Missile Range on September 30th, 2019.

API Reference#

An in-depth explanation of all the functions, classes, etc. that are implemented as part of this library.

esis

Model the ESIS optical system and invert images captured during flight.
